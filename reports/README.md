# Отчёты по практике

Итоговый отчёт по проектной (учебной) практике размещён в двух форматах:

- [report.docx](report.docx) — отчёт в формате Word.
- [report.pdf](report.pdf) — отчёт в формате PDF.

Отчёт составлен по структуре, приведённой в шаблоне [practice_report_template.docx](practice_report_template.docx), и описывает ход и результаты практики: настройку Git и репозитория, подготовку документации, разработку статического сайта проекта и оформление отчётности.
