## Этап 1. Настройка Git и репозитория

**Статус:** выполнено.

- Установлен и настроен Git (`git config` — имя и почта).
- Создан репозиторий практики на GitHub на основе шаблона кафедры (форк), склонирован локально (`git clone`).
- В папку `src/` добавлен исходный код Telegram-бота из проекта по «Проектной деятельности»; модуль работы с API вынесен в `src/mospolytech_api/`.
- Настроен `.gitignore`: исключены секреты и локальные данные (`.env`, `hash_salt.txt`, `bot_data.db`), служебные файлы Python и IDE.
- Изменения зафиксированы коммитами с осмысленными сообщениями и отправлены на GitHub (`git add`, `git commit`, `git push`).
- Освоена работа с ветками: создание, переключение, слияние (`git merge`) и удаление.

**Результат:** репозиторий с корректной структурой размещён на GitHub, секреты в систему контроля версий не попадают.
