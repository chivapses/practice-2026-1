Ресурсы

Полезные ссылки по проекту и использованным технологиям.

Проект и вуз

  • Репозиторий практики на GitHub Исходный код бота, документация и материалы практики.
  • Московский политехнический университет Официальный сайт вуза.
  • Сервис расписания rasp.dmami.ru Официальный источник расписания, откуда бот получает данные.

Технологии бота

  • Python 3 Язык разработки бота.
  • python-telegram-bot Библиотека для работы с Telegram Bot API (long polling).
  • Telegram Bot API Официальная документация по ботам Telegram.
  • SQLite Хранение настроек пользователей (группа, оповещения).

  • Markdown Guide Справочник по синтаксису Markdown для документации.
