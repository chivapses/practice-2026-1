Журнал

Ход работы над практикой по этапам.

Этап 1

Настройка Git и репозитория

Установил и настроил Git, создал репозиторий практики на основе шаблона кафедры. Перенёс код бота в папку src/, настроил .gitignore для исключения секретов, освоил коммиты и работу с ветками.

Этап 2

Документация в Markdown

Изучил синтаксис Markdown и подготовил документацию по практике в папке docs/: обзорный файл и подробный журнал прогресса по этапам.

Этап 3

Разработка сайта

Сверстал статический сайт проекта на чистом HTML и CSS: пять страниц с описанием бота, светлым оформлением, боковым меню и адаптацией под телефоны. Этот сайт — результат данного этапа.

Этап 4

Отчёт по практике

Подготовлен итоговый отчёт по шаблону кафедры в форматах DOCX и PDF, размещён в репозитории в папке reports/ и загружен в СДО (LMS).
